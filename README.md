# Employee Leave Management System

A full-stack web application built using **React** and **Firebase**.

## Features
Employee Registration & Login (Firebase Authentication)
Apply Leave with From & To Date
Leave History with PDF Download
Team Leave Calendar
Public Holidays Display
Firebase Firestore Database
Responsive UI using Bootstrap
Deployed on Netlify

## Tech Stack
React
Firebase (Auth & Firestore)
Bootstrap
Netlify
